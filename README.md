FinAgent is a safety-aware autonomous financial web action agent designed for the FinAgent Hackathon at Techfest, IIT Bombay.

## Project Objective
The goal of this project is to build an intelligent browser-based agent capable of:
- Understanding natural language financial commands
- Planning multi-step workflows
- Executing actions autonomously on websites
- Enforcing a Conscious Pause before all high-risk financial operations

## Technologies (Planned)
- Backend: Flask
- UI: Streamlit
- Browser Automation: Playwright
- AI Model: Large Language Model (LLM)
- Vision Backup: Vision-Language Model (VLM)

## Current Status
This repository currently contains the system architecture and workflow design as part of Phase 1 abstract submission. Full implementation will be developed in Phase 2.
